Project 15 · Sustainable Finance Analytics

Impact Investing Portfolio Optimisation

Balancing Financial Returns and Social Impact Metrics

Languages: Python & R Category: Sustainable Finance Analytics
Research Question Can a portfolio be optimised to maximise social/environmental impact without sacrificing risk-adjusted financial returns?
Methodology

Multi-objective portfolio optimisation combining Markowitz mean-variance framework with impact constraints. Uses the efficient frontier approach with an additional impact dimension, computing the impact-adjusted Sharpe ratio to evaluate trade-offs.

Data Collection & Sources
Source: Yahoo Finance (ESG fund returns via yfinance)
Funds: SUSA, ESGU, DSI, ICLN, VSGX (ESG/impact ETFs) + SPY, AGG (benchmarks)
Period: 2018–2024 daily returns
Key finding: Impact funds show Sharpe ratio ~0.42 vs ~0.36 for benchmarks
Key Methods
Mean-Variance Optimisation
Markowitz framework: maximise expected return for a given risk level — the efficient frontier maps all optimal portfolios
Impact Constraints
Additional optimisation constraint requiring minimum impact score — creates an 'impact-efficient frontier'
Sharpe Ratio Comparison
Risk-adjusted return metric comparing impact portfolios vs conventional benchmarks — tests the 'doing well by doing good' hypothesis
Key Findings
1
Impact fund portfolio achieves a Sharpe ratio of ~0.42 vs ~0.36 for conventional benchmarks, suggesting no performance penalty
2
The impact-efficient frontier lies close to (and sometimes above) the conventional frontier, especially for moderate risk levels
3
Maximum drawdown for impact portfolios was slightly lower during COVID-19, consistent with ESG 'downside protection' theory

import numpy as np
from scipy.optimize import minimize

def neg_sharpe(weights, returns, cov_matrix, rf=0.02):
    port_ret = np.dot(weights, returns.mean()) * 252
    port_vol = np.sqrt(np.dot(weights, cov_matrix @ weights) * 252)
    return -(port_ret - rf) / port_vol
