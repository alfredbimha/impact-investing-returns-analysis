Understanding Impact Investing Returns

Do ESG-focused funds sacrifice financial returns? The evidence says: not much.

The Core Question

Many investors worry that "doing good" means "earning less." We test this by comparing risk-adjusted returns of ESG ETFs against conventional benchmarks.

Sharpe Ratio

Return per unit of risk. Higher = better. Accounts for both returns AND volatility.

Sortino Ratio

Like Sharpe but only penalizes downside risk. Better for loss-averse investors.

Max Drawdown

Worst peak-to-trough decline. How much could you have lost?

Mean-Variance Optimization

We construct an "efficient frontier" — the set of portfolios that offer the highest return for each level of risk. If ESG funds appear on or near this frontier, they don't sacrifice returns.

Maximize: E[R] − (λ/2) × σ² subject to: weights sum to 1, no shorting
Finding: ESG funds show competitive Sharpe ratios. The efficient frontier including ESG funds is nearly identical to the one without — suggesting minimal return sacrifice.
