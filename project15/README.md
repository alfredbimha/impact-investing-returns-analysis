# Impact Investing Returns Analysis

## Research Question
Do impact/ESG-focused funds sacrifice financial returns?

## Methodology
**Language:** Python  
**Methods:** Performance analytics, mean-variance optimization

## Data
Yahoo Finance — 8 ESG ETFs vs conventional benchmarks (2018–2025)

## Key Findings
ESG funds show competitive risk-adjusted returns; efficient frontier analysis suggests minimal return sacrifice.

## How to Run
```bash
pip install -r requirements.txt
python code/project15_*.py
```
